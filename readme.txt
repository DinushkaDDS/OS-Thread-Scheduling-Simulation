1. Open the project file using Visual Studio. We have used version 2015.
2. Then start project.
3. Using the process table on the top left corner, give the values of arrival and service times. 	
   Since the space is less to display the process timeline, we have restricted it to 20. So be sure to give      keep the total service time uper limt as 20 cycles.  
4. You wil see the form with start button. Press it to start the process.
5. To proceed to next process press continue in MessageBox

Or direct to OSProject\OSProject\bin\Debug and double click on OSProject.exe to just run the application.
